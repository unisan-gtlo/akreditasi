Login & Akun

Saya lupa password, apa yang harus dilakukan? ›
Reset password hanya bisa dilakukan oleh admin Pustikom. Kirim email ke pustikom@unisan-g.id dengan subjek "Reset Password SIAKRED", sebutkan username/NIDN Anda. Admin akan reset password dan kirim password baru via email resmi.
Akun saya terkunci, bagaimana cara buka? ›
Akun terkunci 15 menit setelah 5 kali gagal login berturut-turut. Tunggu 15 menit, akun otomatis bisa login lagi. Kalau tidak mau menunggu, hubungi admin Pustikom untuk unlock manual.
Kenapa muncul CAPTCHA saat login? ›
CAPTCHA muncul setelah 2 kali gagal login untuk mencegah brute force attack. Isi sesuai pertanyaan matematika yang muncul, lalu login normal.

Upload Dokumen

File saya tidak bisa di-upload, muncul error ›
Cek beberapa hal:
  • Ukuran file maksimal 20 MB - kalau lebih, compress dulu atau gunakan link Google Drive
  • Format yang diterima: PDF, DOCX, XLSX, PPTX, PNG, JPG
  • Nama file tidak boleh mengandung karakter spesial seperti / \ : * ? " < > |
  • Pastikan koneksi internet stabil
Saya upload dokumen tapi tidak muncul di daftar ›
Coba refresh halaman (F5 atau Ctrl+R). Kalau masih tidak muncul, cek apakah upload-nya benar masuk dengan buka menu Dokumen Saya. Kalau tidak ada juga, hubungi admin dengan menyertakan screenshot.
Bagaimana cara ganti file yang sudah di-upload? ›
Gunakan fitur Upload Revisi di halaman detail dokumen. Setiap revisi akan dinomori (r2, r3, dst). File lama TIDAK terhapus tapi menjadi history. Verifikator akan review revisi terbaru.

Verifikasi & Notifikasi

Kenapa dokumen saya belum di-review setelah berhari-hari? ›
LP3M biasanya review dokumen berdasarkan deadline sesi. Kalau sudah lebih dari 1 minggu, Anda bisa:
  • Ingatkan LP3M via email atau WA
  • Hubungi Dekan fakultas Anda (yang juga bisa review)
  • Cek deadline sesi di menu Sesi Akreditasi - jangan tunggu sampai mepet
Notifikasi tidak muncul di bell icon padahal ada aktivitas ›
Coba refresh halaman terlebih dahulu. Kalau masih tidak muncul, mungkin Anda bukan penerima notif-nya (notif hanya ke uploader dokumen, bukan ke verifikator sendiri). Cek halaman Notifikasi lengkap untuk lihat semua history.
Perbedaan Rejected vs Need Revision? ›
Rejected = dokumen ditolak total, Anda perlu upload file baru dari awal (biasanya karena salah butir atau file sama sekali tidak relevan). Need Revision = dokumen basic-nya OK tapi perlu perbaikan, Anda upload versi revisi (r2) dengan fix sesuai catatan verifikator.

Bundle & Share Asesor

Asesor bilang link bundle tidak bisa dibuka ›
Cek status token di halaman Bundle:
  • Apakah token masih Aktif? Kalau sudah di-revoke, buat token baru
  • Apakah tanggal expired sudah lewat? Kalau iya, buat token baru dengan expiry lebih panjang
  • Pastikan asesor copy link lengkap (tidak ada spasi/karakter terpotong)
Kenapa beberapa dokumen tidak muncul di bundle public asesor? ›
Bundle Public hanya menampilkan dokumen berstatus APPROVED. Dokumen Pending/Rejected/Need Revision tidak tampil ke asesor. Pastikan LP3M sudah verifikasi semua dokumen sebelum share token ke asesor.
