1. Login ke SIAKRED

Gunakan kredensial kampus UNISAN Anda untuk masuk ke sistem.

1Buka browser, kunjungi https://siakred.unisan-g.id/login/
2Masukkan username & password UNISAN Anda
3Klik tombol Login
Tips: Kalau gagal 3 kali berturut-turut, sistem akan meminta CAPTCHA. Kalau gagal 5 kali, akun dikunci sementara 15 menit. Kalau lupa password, hubungi admin Pustikom.

2. Upload Dokumen Pertama Kali

Setelah login, Anda akan melihat daftar butir dokumen yang menjadi tanggung jawab prodi Anda.

1Klik menu sidebar Dokumen » Butir Saya
2Pilih butir yang ingin diisi (misal EM-1.1-P), klik Upload
3Isi form:
  • Judul Dokumen - nama jelas dokumen (wajib)
  • Deskripsi - ringkasan singkat (opsional)
  • Tahun Akademik - pilih tahun relevan (misal 2025/2026)
  • File - PDF/DOCX/XLSX (max 20 MB)
4Klik Submit Upload
Format file yang diterima: PDF (diutamakan), DOCX, XLSX, PPTX, PNG, JPG. Ukuran maksimal 20 MB per file.

Setelah upload berhasil, dokumen otomatis masuk status Pending - menunggu review LP3M/Dekan.

3. Upload Revisi Dokumen

Kalau dokumen Anda mendapat feedback "Perlu Revisi" dari verifikator, Anda perlu upload versi perbaikan.

1Buka dokumen yang butuh revisi (dari menu Dokumen Saya atau dari link di notifikasi)
2Baca catatan verifikator dengan teliti di bagian "Riwayat Verifikasi"
3Perbaiki dokumen Anda sesuai catatan
4Klik tombol Upload Revisi, pilih file baru, dan beri catatan singkat tentang perubahan
5Submit - dokumen kembali ke status Pending untuk di-review ulang
Penting: Setiap revisi akan dinomori (r1, r2, r3, dst). File lama tidak dihapus tetapi menjadi history. Verifikator akan review berdasarkan revisi terbaru.

4. Cek Status Verifikasi

Setiap dokumen punya 4 status:

  • 🔵 Pending - Menunggu review verifikator
  • 🟢 Approved - Disetujui, masuk Bundle Export untuk asesor
  • 🔴 Rejected - Ditolak, perlu review ulang dari awal
  • 🟠 Need Revision - Perlu perbaikan sesuai catatan

Cek status via:

  • Menu sidebar Dokumen Saya - lihat semua dokumen Anda dengan status
  • Dashboard utama - summary progress
  • Notifikasi - saat ada perubahan status, auto terkirim

5. Baca Notifikasi Feedback

Setiap kali verifikator approve/reject dokumen Anda, notifikasi otomatis terkirim.

1Lihat bell icon di topbar kanan - kalau ada badge merah angka, artinya ada notifikasi baru
2Klik bell - dropdown muncul dengan 5 notifikasi terakhir
3Klik notifikasi untuk lihat detail + baca catatan verifikator
4Kalau perlu lihat history lengkap, klik "Lihat Semua Notifikasi" di footer dropdown

6. Tips & Trik

  • Upload lebih awal - jangan nunggu deadline, biar ada waktu revisi kalau ada feedback
  • Beri judul yang jelas - "VMTS Prodi Teknik Elektro 2025" lebih baik daripada "VMTS"
  • Baca catatan verifikator dengan teliti - kebanyakan revisi ditolak karena tidak mengikuti catatan
  • Gunakan format PDF - lebih profesional dan konsisten tampilan di semua device
  • Scan dengan resolusi wajar - jangan terlalu tinggi (bikin file besar), tidak terlalu rendah (tulisan tidak jelas)
